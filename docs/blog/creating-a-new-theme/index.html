このサイトの目的は何かね？

motivation of this site

January 24, 2018 仁志

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Email

皆さん初めまして、私、「仁志」と申します。

このブログの目的から書いていこうと思います。

皆さんは、どんな勉強法を使って、勉強をしていますか？

「勉強法？そんなの考えたことないや。」

「我武者羅にやっている。」

「なかなか良い方法がない。」

自分に合った勉強法が見つからないと悩んでいる方々は多いと思います。

学生時代、僕は色々な勉強法を考え、試してきました。今は、社会人になり、自分に出来ることは何なのかを考えるようになりました。

そして、辿りついた答えが自分が試してきた勉強法を公開し社会に還元した方が良いのではないかでした。

そしてこのブログを立ち上げました。

主に学生の方向けの内容になりますが、お子さんの勉強に悩んでいる保護者の方や、社会人で資格勉強をされている方の参考にもなればと思っております。

ただ、僕が試してきた勉強法を使っても、もしかしたら、全く効果がない人がいるかもしれません。

それでも、ほんの少しだけ、成績が上がったり、力がつくお役にたてればと思っております。

あくまで、このブログの勉強法を参考にしていただいて

「こんなやり方があるんだ。これは、使えそうだな！」

「この方法をこうするともっと効率が良くなるな☆」

等、自分なりに考えてご利用いただければと思っております。よろしくお願いします。（＾０＾）V

それでは、まとめに入ります。

☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆☆

このブログの目的は何かね？

～多くの人に勉強法を還元することにあります。～

仁志
